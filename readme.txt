== Description ==
Make slideshows with multilayer facility and customize them with easy options

Features 

* Responsive
* Multi Layer
* Lots of animation options
* Elements can be placed by drag and drop
* Slideshow is also working in widgets

== Installation ==


1. Upload `wp-animate-slideshow` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
